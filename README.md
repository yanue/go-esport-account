# go-esport-account

### 包管理
1. 根据go.mod文件下载依赖包
> go mod download
* 或者直接
> go build

2. 更新已存在的包
> go get -u

3. 使用代理下载
> proxifier + shadowsocks
* 或者
> vpn

### go-micro
1. 命令行
> github.com/micro/micro

2. 微服务
> github.com/micro/go-micro

### 其他基础包
1. 数据库操作
> github.com/jinzhu/gorm

2. redis操作
> github.com/go-redis/redis

3. http框架
> github.com/gin-gonic/gin
